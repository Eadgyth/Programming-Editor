Features

Text editing

  • Showing files in tabs. The tabbar may be hidden if only one document is open.
  • Enabling and disabling word-wrapping.
  • Undo/redo.
  • A basic find/replace function.
  • Clearing end-of-line (trailing) white spaces.
  • An "exchange editor" panel to edit text in a separate view and to facilitate the exchange of text within a file or between files
  • A basic syntax highlighting for java, perl, html, javascript and CSS.
  • Block-wise increase or decrease of the indentation.
  • A basic auto-indentation.

Testing source code

  • A simple and flexible assignment of files as coding project.
    • A project may be simply the location of a source file.
    • The program also works with a given directory structure of a project. For Java this may include folders for source files, executable class files and packages.
    • A number of projects can be defined and one can easily switch between projects.
    • Retrieval of already defined projects after newly starting the program.
  • Compiling and testing Java code and bundling a Java program in an executable jar file. Non-Java files used as resources can be included.
  • Testing a Perl script.
  • Viewing Html code in the default web browser.
  • A basic (interactive) console to view the output (error messages) during compiling and testing a Java or Perl program.

Other

  • A basic file explorer for projects.
  • Running self-chosen system commands in the current working directory.
